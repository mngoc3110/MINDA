# Danh sách Tính năng Hệ thống MINDA (Feature Requirements)

## 1. Học sinh (Student)
- Học tập: Xem Video, Tham gia Live-class (WebRTC), Làm bài tập trắc nghiệm/tự luận.
- **Phòng thí nghiệm Toán ảo (3D Math Labs):** Tương tác xoay, phóng to, thu nhỏ các mô hình hình học không gian (Solid Geometry), mặt cầu, đồ thị phương trình 3D trực tiếp trên web bằng công nghệ WebGL (React Three Fiber).
- Tính năng AI: Tự động gợi ý nhắc lại (Spaced repetition), Nhận diện độ tập trung.
- Gamification: Tích lũy EXP, Lên cấp Rank (Đồng -> Thách Đấu), Bảng Leaderboard, Nhận huy hiệu (Badge).
- Tài khoản: "Bộ nhớ học tập" (Cloud Drive mini để lưu trữ tài liệu, tập tin nộp bài).

## 2. Giáo viên (Teacher / Instructor)
- Quản lý Lớp & Khóa học: Đăng tải video/tài liệu bài giảng, Tạo đề thi.
- Live-class: Điều khiển lớp Live, Mute học sinh, Chức năng giơ tay.
- Analytics & AI Dashboard (RAPT-CLIP AI):
  - Báo cáo chú ý: "Cả lớp mất tập trung vào phút thứ...".
  - Báo cáo cá nhân: Đo hiệu suất của riêng một học sinh nào đó.
- Auto-grading: Hỗ trợ chấm điểm thủ công xen lẫn tự động.

## 3. Quản trị viên (Admin)
- Quản lý cốt lõi: Tạo/Xóa tài khoản, Phân quyền.
- Quản lý Tài chính: Quản lý học phí, Báo cáo Doanh thu.
- Setup Hệ thống: Cấu hình Server Model AI, Băng thông mạng.
- Security: Logs, Chống DDoS, Xuất File lưu trữ.
